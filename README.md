# C Compiler  
A C compiler developed using lex and yacc  
  
## Phases implemented:  
1. Lexical Analysis  
2. Syntax Analysis  
3. Semantic Analysis  
4. Intermediate Code Generation (ICG)  
  
## Features:  
* Symbol Table generation
* Intermediate code is stored in Quadruples format
